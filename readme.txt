VSCode Plugins used for angular development:
1) Angular Snippets: Jogn Papa
2) Angular files (Quickly scaffold angular file templates)
3) Angular Language service
4) Auto rename tag (to keep the opening and closing tag name in sync)
5) Bracket pair colorizer (same color for opening and closing brackets)
6) Debugger for Chrome
7) Material Icon Theme (just to make VSCode more pleasing to work with material type icons)
8) Prettier code formatter (html, css and js code formatter)
9) TSLint (for custom rules)
10) Angular2-switcher (to switch between css, html and ts files easily)
